# Vent

Defaults to `client` command if none is provided

```
vent creates a websocket tunnel between among us instances

Usage:
  vent [command]

Available Commands:
  client      Run client side of the tunnel
  help        Help about any command
  host        Run host side of the tunnel

Flags:
      --colors          Log output with colors (default true)
      --game-port int   Game server port (default 22023)
  -h, --help            help for vent
      --log string      The log level to output (default "info")
      --port int        Websocket port (default 56217)

Use "vent [command] --help" for more information about a command.
```
